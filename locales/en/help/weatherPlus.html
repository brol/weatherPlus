About

This plugin displays the weather for one or several cities via webservice Weather.com.

Configuration

The weather service issued by Weather.com's free. However, it is need to register with Weather.com to fully benefit legally webservice.

To save you from Weather.com, go to the site, create your free account and activate it. Log in to the website and fill in the form (Key Settings).
See the information Key ID and Project Name in the plugin configuration form and save.

Search a city

Enter a city name to see if it is recognized by the webservice. For the definition of cities in the associated widget plugin, you indicate the choice of valid names of cities or identifiers of cities (city ID) unique.

Weather Widget

In the presentation widgets, place the Weather widget in one component. Then enter valid names of cities (you can search City since the tab dedicated to the plugin administration) or the cities identifiers (city ID) unique separated by a comma. It is possible to display the local time of the selected cities (with the restriction of one clock per time zone).
