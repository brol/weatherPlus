À propos

Ce plugin permet d'afficher la météo d'une ou de plusieurs villes via le webservice Weather.com.

Configuration

Le service météo délivré par Weather.com est gratuit. Il est cependant nécessaire de s'enregistrer auprès de Weather.com pour bénéficier pleinement et légalement du webservice.

Pour vous enregistrer auprès de Weather.com, rendez-vous sur le site, créez votre compte gratuit et activez-le. Identifiez-vous sur le site et renseignez le formulaire (Key Settings).
Reportez les informations Key ID et Project Name dans le formulaire de configuration du plugin et enregistrez.

Recherche d'une ville

Entrez le nom d'une ville pour vérifier si elle est bien reconnue par le webservice. Pour la définition des villes dans le widget associé au plugin, vous indiquerez au choix des noms valides de villes ou bien des identifiants de villes (ID de ville) uniques.

Widget Météo

Dans les widgets de présentation, placez le widget Météo dans l'un des volets. Indiquez ensuite des noms valides de villes (vous pouvez faire une recherche de ville depuis l'onglet dédié de l'administration du plugin) ou bien des identifiants de villes (ID de ville) uniques en les séparant par une virgule. Il est possible d'afficher l'heure locale des villes sélectionnées (avec la restriction d'une seule horloge par fuseau horaire).
